## ğŸš€ Fitur Utama

Aplikasi ini memiliki dua sisi pengguna: **Pengunjung (Public)** dan **Administrator**.

### ğŸ”¹ Halaman Publik (Frontend)
- **Galeri Produk:** Katalog visual berbagai jenis kubah (Enamel, Galvalum, Stainless Gold, dll).
- **Galeri Promosi:** Halaman khusus untuk menampilkan promo atau proyek terbaru yang sedang berjalan.
- **Tentang Kami & Produk:** Informasi detail mengenai profil perusahaan dan spesifikasi bahan kubah.
- **WhatsApp Integration:** Tombol "Hubungi Kami" yang langsung terhubung ke chat WhatsApp admin untuk pemesanan cepat.
- **Responsive Design:** Tampilan rapi di HP maupun Laptop berkat Bootstrap.

### ğŸ”¹ Halaman Admin (Backend)
- **Dashboard:** Ringkasan data (jumlah produk, galeri, dll).
- **Manajemen Produk:** Tambah, edit, dan hapus data produk kubah.
- **Manajemen Galeri:** Upload foto-foto dokumentasi proyek.
- **Manajemen Konten:** Edit informasi promo dan profil perusahaan tanpa coding.
